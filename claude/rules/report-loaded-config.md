# Report Loaded Configuration

- At the start of each conversation, list loaded config files, plugins, plugin context, and active skills in a bordered table
- Annotate skills with source (plugin name or "built-in"); annotate plugin context files with plugin name
- Omit sections with no entries
- Keep brief (names only), then proceed with the user's request

Example:
```
+----------------------------------+
|  Configuration Files Loaded      |
|  - file1.md                      |
+----------------------------------+
|  Active Skills                   |
|  - skill-name (built-in)         |
+----------------------------------+
```
